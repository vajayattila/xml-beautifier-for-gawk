Welcome to the XML beautifier for Gawk wiki!

Introduction

The XML beautifier for Gawk is an awk script for formating xml files.

Get Gawk

The script is required to run a version of gawk. This can be obtained at the following locations:

for Windows: http://gnuwin32.sourceforge.net/packages/gawk.htm

for Others: https://www.gnu.org/software/gawk/

Run the formatxml.awk (on windows)

Sample parameters for the format example.xml. The script print the formated xml to screen:

gawk -f formatxml.asw example.xml

The formated xml redirection to a file:

gawk -f formatxml.asw example.xml > formated.xml

You can read formating results in the xmlinfo.txt
